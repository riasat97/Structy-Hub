## is palindrome

Write a function, _is\_palindrome_, that takes in a string and returns a boolean indicating whether or
not the string is the same forwards and backwards.

#### test_00:

```python
is_palindrome("pop") # -> True
```

#### test_01:

```python
is_palindrome("kayak") # -> True
```

#### test_02:

```python
is_palindrome("pops") # -> False
```

#### test_03:

```python
is_palindrome("boot") # -> False
```

#### test_04:

```python
is_palindrome("rotator") # -> True
```

#### test_05:

```python
is_palindrome("abcbca") # -> False
```

#### test_06:

```python
is_palindrome("") # -> True
```

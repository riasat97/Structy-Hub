## all unique

Write a function, _all\_unique_, that takes in a list. The function should return a boolean
indicating whether or not the list contains unique items.

#### test_00:

```python
all_unique(["q", "r", "s", "a"]) # -> True
```

#### test_01:

```python
all_unique(["q", "r", "s", "a", "r", "z"]) # -> False
```

#### test_02:

```python
all_unique(["red", "blue", "yellow", "green", "orange"]) # -> True
```

#### test_03:

```python
all_unique(["cat", "cat", "dog"]) # -> False
```

#### test_04:

```python
all_unique(["a", "u", "t", "u", "m", "n"]) # -> False
```

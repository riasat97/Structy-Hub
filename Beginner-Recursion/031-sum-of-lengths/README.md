## sum of lengths

Write a function _sumOfLengths_ that takes in a list of strings and returns the total length of the strings.

You must solve this recursively.

#### test_00:

```python
sum_of_lengths(['goat', 'cat', 'purple']) # -> 13
```

#### test_01:
```python
sum_of_lengths(['bike', 'at', 'pencils', 'phone']) # -> 18
```

#### test_02:
```python
sum_of_lengths([]) # -> 0
```

#### test_03:
```python
sum_of_lengths(['', ' ', '  ', '   ', '    ', '     ']) # -> 15
```

#### test_04:
```python
sum_of_lengths(['0', '313', '1234567890']) # -> 14 
```
